# Дипломный Проект «Разработка образовательного симулятора управления робомышью в лабиринте на платформе Unity»

Проект представляет собой систему визуального программирования, разработанную в **Unity**, которая позволяет управлять роботом с помощью цепочки логических блоков.

## 📌 Описание проекта

Данный проект реализует:
- режим ручного управления роботом (WASD);
- управление робот при помощи кода(API);
- визуальное программирование на основе блоков;
- соединение блоков в последовательную цепочку выполнения;
- взаимодействие с роботом через API;
- сохранение данных в базе данных;
- механизм **Replay** для повторного воспроизведения программы.

Проект разрабатывался как **курсовая работа**.

## 🧱 Архитектура проекта

### 🔹 Блоки (Blocks)
Каждый блок представляет отдельное действие:
- движение
- поворот
- ожидание
- старт / стоп

Блоки:
- имеют ссылку на следующий блок;
- выполняются последовательно;

---

### 🔹 Связь блоков
Блоки соединяются в цепочку (Block Chain).  
Это позволяет:
- гибко управлять логикой;
- легко изменять порядок выполнения;

---

### 🔹 WorkspaceProgramRunner
Отвечает за:
- поиск стартового блока;
- запуск выполнения;
- последовательное выполнение блоков;
- контроль и логирование процесса.

---

### 🔹 BlockChainManager
Управляет:
- созданием блоков;
- удалением блоков;
- связями между блоками.

---

### 🔹 API Controller
Используется для:
- отправки команд роботу;
- получения ответов;
- обработки ошибок.

---

### 🔹 База данных
БД используется для:
- хранения попыток;
- реализации механизма replay.

---

## 🔁 Replay

Replay реализован за счёт:
- сохранения порядка выполнения команд;
- хранения параметров каждого лабиринта;
- повторного запуска цепочки выполнения;
- запоминание seed.

Это позволяет:
- воспроизводить ранее созданные попытки;
- анализировать и отлаживать поведение робота.
